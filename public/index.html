How to deploy
GITStand は API キーを使用しません。リポジトリ URL と OAuth / GitHub アプリ連携で操作します。
# 例: リポジトリ URL を使ってデプロイをトリガーする（認証は OAuth）
POST https://api.gitstand.example/v1/deploy
Content-Type: application/json

{
  "owner": "username",
  "repo": "repository",
  "branch": "main"
}

    • Getting started

    Connect your GitHub repository via OAuth or GitHub App, configure build settings, and deploy.

    Authentication

    GITStand は API キーを使用しません。ユーザー認証とリポジトリアクセスは GitHub OAuth または GitHub App を通じて行います。

    Deploy API

    リポジトリ情報を送るだけでデプロイをトリガーできます。レスポンスにログ URL が含まれます。

    Environment Variables

    プロジェクト単位でシークレットを管理し、UI とログではマスクされます。

    Webhooks

    デプロイイベントを受け取るために Webhook を設定できます。署名シークレットで検証してください。
